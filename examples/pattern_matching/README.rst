Pattern matching
================

Use pattern matching to promote an object to its class from one of its
traits. Pattern matching can match object contents or value as well.

.. code-block::

   $ mys run
   Class Foo with a=1 and b="hi".
   Class Foo with a=2 and b="ho".
   Class Foo.
   Class Bar.
   Other class: Fie()
   Zero integer.
   Five integer.
   String foo.
   Other string: bar
