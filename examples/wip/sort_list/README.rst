Sort a list
===========

An example of how to sort a list.

.. code-block::

   $ mys run
   Persons:
   [
       Person("Erik", 31),
       Person("Linda", 3),
       Person("Frida", 76),
       Person("Kalle", 77),
   ]
   Sorted by name:
   [
       Person("Erik", 31),
       Person("Frida", 76),
       Person("Kalle", 77),
       Person("Linda", 3)
   ]
   Sorted by age:
   [
       Person("Linda", 3),
       Person("Erik", 31),
       Person("Frida", 76),
       Person("Kalle", 77)
   ]
